# Alu-Fab Pakistan — Premium Website Implementation Plan

## Structure
- `index.html` — Main HTML file with all sections
- `css/style.css` — Custom CSS (animations, glassmorphism, layout)
- `js/main.js` — JavaScript (scroll effects, counters, modals, lightbox, carousel, FAQ, dark mode)
- Assets generated via AI images

12. Contact (form + map + info)
13. Footer
